Specifications

File Delivery

  • MMT File upload - Upload artwork
  • 3rd party file transfer services - mmtgraphics@mmt.com
  • E-mail files and project information - mmtgraphics@mmt.com

File Preparation

  • Preferred file format - PDF
    • Recommended PDF preset - PDF/x-4
  • Image Design Applications - .PSD / .PSB / .EPS / .TIF / .JPEG
    • CMYK format
    • Convert all fonts to outlines
  • Layout Design Applications - Adobe InDesign/Adobe Illustrator/QuarkXPress
    • Layout Document - .INDD / .AI / .QXP
    • Include all linked image files used in layout or ensure they are embedded.
    • Include all fonts or preferrably convert all fonts to outlines

Color

  • MMT is a G7 certified master printer. All printing is in the CMYK colorspace.
  • Recommended CMYK profile - GRACoL
